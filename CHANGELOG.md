# Changelog

All notable changes to SSmooth are documented here.

## v1.0.1

First working public release. Supersedes the initial v1.0.0 tag, which pointed at an early build that did not load.

### Added
- Read only system analysis: hardware, storage breakdown, memory pressure, top processes, and startup items.
- Diagnosis that ranks likely causes of slowness by impact, each with a plain language explanation, estimated gain, and risk label.
- Safe, reversible cleanup: per item and per category toggles, a dry run preview, Trash by default, and a full action log.
- Results view with before and after disk usage.
- Personalized maintenance plan, derived entirely from your own scan, with each tip tied to a real measurement.
- Apple and Teenage Engineering inspired design with full light and dark themes.
- Continuous integration that builds and tests on macOS (Apple Silicon and Intel) and Windows.

### Notes
- macOS builds are not yet code signed. On first launch, right click the app and choose Open.
